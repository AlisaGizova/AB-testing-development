# Разработка A/B-тестирования и анализ результатов

**Задачи проекта:** Оценить влияние нового алгоритма рекомендательной системы на ключевые метрики вовлеченности пользователей в развлекательном приложении с помощью корректно спланированного и проведенного A/B-теста.

**Описание проекта:** Проведен анализ A/B-теста нового алгоритма рекомендаций контента. Рассчитаны параметры и оценена корректность проведения эксперимента. Проанализировано влияние на ключевую метрику — долю успешных сессий (просмотр ≥4 страниц). Определена статистическая значимость результатов с использованием соответствующих статистических критериев. Подготовлены рекомендации по внедрению новой системы.
